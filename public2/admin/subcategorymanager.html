사용방법 (검색/수정)

1) 모드 선택: 모달 안에서 푸드/통합 선택
2) 사업자번호만으로 검색 가능: 숫자만 입력해도 됨
3) 상호명 검색: 검색 결과에 “몇 페이지/몇 번째” 표시
4) 수정: 슬롯 클릭 → 모달에서 방식 선택(버튼) → 저장
※ 저장 API: /admin/subcategory/update (404면 모달 상태에 에러 표시)
