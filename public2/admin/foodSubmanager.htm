    page=subcategory_food / position=subcategory_food_new

사용 방법 메모

  1. 오른쪽 미리보기에서 원하는 슬롯 영역을 클릭합니다.
  2. 아래 “선택된 슬롯 정보” 카드에 표시되는 page / position 값을 확인합니다.
  3. 이미지 업로드/링크 설정은 공통 레이아웃 광고/이미지 관리 화면에서 같은 page, position 으로 등록합니다.
